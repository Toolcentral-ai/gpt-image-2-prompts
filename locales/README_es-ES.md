[English](../README.md)

Este repositorio organiza plantillas de prompts para GPT Image 2 y metadatos de categorías para flujos de generación de imágenes, navegación de prompts, búsqueda de casos y agent skills.

Sitio principal de prompts: https://model-prompt-lab.com/models/gpt-image-2/prompts

Los enlaces de categoría usan `q=<category>` en lugar del formato anterior `categories=<category>`. La búsqueda de Model Prompt Lab entiende estos términos semánticamente y recupera casos de GPT Image 2 relacionados.
