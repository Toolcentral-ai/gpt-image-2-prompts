### What is included

- Reusable GPT Image 2 prompt text
- Detailed source category links from the source README
- Featured prompt index and high-quality images under `assets/featured/`
- Multi-language README entry files modeled after the source repository
- Prompt variables parsed from `{argument name="..." default="..."}` blocks
- Category, group, tag, input-mode, reference-image, and attribution metadata

### Data files

- `data/image-prompt-templates.json` — full enriched template bundle
- `data/categories.json` — source category groups and gallery links
- `data/featured-prompts.json` — featured prompts from the source README
- `data/source-prompts.json` — source README prompt index
- `docs/source-gallery.md` — bilingual category and featured-prompt navigation
- `skills/image-prompt-templates/SKILL.md` — bilingual agent skill
